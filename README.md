# 🗂️ Python Automated Backup Script

A simple yet effective Python script that **automates folder backups** into compressed `.tar.gz` archives with filenames based on the current date.
---

## 🚀 Features

- 📦 Compresses and archives any folder
- 🗓️ Names the backup file using today’s date
- 🪟 Optimized for Windows systems
- 🧩 Uses only built-in Python modules (`shutil`, `datetime`, `os`)
